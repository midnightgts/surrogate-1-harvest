### Implementation Plan
1. **Identify the relevant script**: Locate the training script that is currently being used to fetch data from the HF API.
2. **Modify the script**: Update the script to use the `list_repo_tree` method with `recursive=False` to fetch the list of file paths for a specific date folder.
3. **Save the file list**: Save the list of file paths to a JSON file.
4. **Embed the file list**: Embed the JSON file containing the file list in the training script.
5. **Use CDN-only fetches**: Update the training script to use CDN-only fetches with zero API calls during data load.

import json
import requests

# Define the repository and date folder
repo = "huggingface/datasets"
date_folder = "2026-04-29"

# Fetch the list of file paths using list_repo_tree
response = requests.get(f"https://huggingface.co/{repo}/tree/main/{date_folder}")
file_list = response.json()

# Save the file list to a JSON file
with open("file_list.json", "w") as f:
    json.dump(file_list, f)

# Embed the file list in the training script
with open("train.py", "r+") as f:
    script = f.read()
    f.seek(0)
    f.write(f"file_list = {json.dumps(file_list)}\n")
    f.write(script)
