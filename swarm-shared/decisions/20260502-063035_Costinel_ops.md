#### Implementation Plan

1. **Update `train.py` to use CDN downloads**: Replace `hf_hub_download` with `requests.get` to download files from CDN directly
2. **Embed file list in training script**: Use a single API call from Mac to `list_repo_tree(path, recursive=False)` for one date folder, save list to JSON, and embed in `train.py`
3. **Verify CDN downloads**: Ensure that CDN downloads are not counted towards HF API rate limit

#### Code Snippets

```python
import requests

# Update train.py to use CDN downloads
def download_file_from_cdn(repo, path):
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
    response = requests.get(url)
    if response.status_code == 200:
        return response.content
    else:
        raise Exception(f"Failed to download file from CDN: {url}")

# Embed file list in training script
def get_file_list(repo, date_folder):
    url = f"https://huggingface.co/api/datasets/{repo}/list_repo_tree/{date_folder}"
    response = requests.get(url)
    if response.status_code == 200:
        return response.json()
    else:
        raise Exception(f"Failed to get file list from API: {url}")
```
