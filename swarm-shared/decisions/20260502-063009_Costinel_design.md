**Task:** Implement HF CDN Bypass for dataset training to avoid API rate limit

**Highest-value incremental improvement:** Bypass HF API rate limit by downloading public dataset files from CDN

**Implementation Plan:**

1. **Identify public dataset files:** Use `list_repo_tree` API to get a list of files in the dataset repository, filtering out private files.
2. **Download files from CDN:** Use `requests` library to download public dataset files from the CDN URL (`https://huggingface.co/datasets/{repo}/resolve/main/{path}`).
3. **Save files to local storage:** Store the downloaded files in a local directory, using a consistent naming convention.
4. **Update training script:** Modify the training script to load the downloaded files from local storage instead of using the HF API.

**Code Snippet:**
```python
import requests

def download_dataset_files(repo, path):
    # Get list of public files in the repository
    files = hf_api.list_repo_tree(repo, path, recursive=False)
    public_files = [f for f in files if not f.private]

    # Download public files from CDN
    for file in public_files:
        url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file.path}"
        response = requests.get(url)
        with open(f"{file.path}", "wb") as f:
            f.write(response.content)

    # Save files to local storage
    local_dir = "data/local"
    os.makedirs(local_dir, exist_ok=True)
    for file in public_files:
        shutil.copy(f"{file.path}", local_dir)

# Update training script to load files from local storage
train_script = """
# Load data from local storage
data = pd.read_parquet(f"{local_dir}/data.parquet")
"""
```
