# airship / ops

**Highest-Value Incremental Improvement:**
**Bypass HF API Rate Limit with CDN**

**Implementation Plan:**

1. **Identify Public Dataset Files**: Use `list_repo_tree(path, recursive=False)` to get a list of public dataset files at `https://huggingface.co/datasets/{repo}/resolve/main/{path}`.
2. **Save List to JSON**: Save the list of public dataset files to a JSON file on the Mac (orchestration script).
3. **Embed in Training Script**: Embed the JSON file in the training script (e.g., `train.py`) to use the CDN tier for dataset downloads.
4. **Lightning Training**: Use Lightning's CDN-only fetches with zero API calls during data load.

**Code Snippets:**

```bash
# Identify public dataset files
list_repo_tree=$(curl -X GET \
  https://huggingface.co/datasets/{repo}/resolve/main/ \
  -H 'Authorization: Bearer YOUR_HF_TOKEN' \
  -H 'Content-Type: application/json')

# Save list to JSON
echo "$list_repo_tree" > public_dataset_files.json

# Embed in training script
train.py:
  import json
  with open('public_dataset_files.json', 'r') as f:
    public_dataset_files = json.load(f)
  # Use public_dataset_files for dataset downloads
```

**Expected Outcome:** Bypass HF API rate limit with CDN tier for public dataset files, reducing the number of API calls and improving training performance.
