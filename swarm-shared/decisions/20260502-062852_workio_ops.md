**Implementation Steps:**

1. **HF CDN Bypass:**
	* Create a new file `cdn_bypass.py` in the `workio/server` directory with the following content:
```python
import requests
import json

def get_file_list(repo, path):
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
    response = requests.get(url)
    return json.loads(response.content)

def download_file(repo, path, filename):
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{path}"
    response = requests.get(url, stream=True)
    with open(filename, 'wb') as f:
        for chunk in response.iter_content(1024):
            f.write(chunk)
```
2. **Update `train.py` to use HF CDN Bypass:**
	* Import the `cdn_bypass` module in `train.py` and use the `get_file_list` and `download_file` functions to download files from the HF CDN.
